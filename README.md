# pd_PasswordManager
GUI Application to Manage Different Application Password 
Download PasswordManager-linux.tar.gz file to on Linux system. To run it, follow the below steps:

1) Open their terminal.

2) Extract the archive:

3) tar -xzvf PasswordManager-linux.tar.gz

4) (Optional) Make the file executable: Sometimes file permissions can be lost. It's good practice to ensure the file is executable.

chmod +x PasswordManager

5) Run the application:

./PasswordManager

Your password manager application will now start, with no other installation steps required.
